This archive contains the DBSCAN Extension for NetLogo, release 0.3. 

Note: This release only supports NetLogo versions 6 and higher. The last release to support NetLogo 5 is 0.2.1.

To install the extension, copy the folder dbscan (not just the content) into NetLogo's app/extensions folder. 
For NetLogo 6.1 onwards, consider using the auto-installation feature (triggered by adding extension header to the model) 
or the Extension Manager.

For more information and detailed installation and usage instructions, refer to https://github.com/chrfrantz/NetLogo-Extension-DBSCAN#readme
